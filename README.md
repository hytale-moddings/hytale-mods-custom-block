# Hytale Mod: Crystal Glow Block (Create A Block)

Companion mod for the [Hytale Modding Manual](https://hytale-moddings.github.io/hytale-modding-docs/) tutorial on **Create A Block**.

**Repository:** [hytale-moddings/hytale-mods-custom-block](https://github.com/hytale-moddings/hytale-mods-custom-block)

## What This Mod Does

Adds a custom **Crystal Glow Block** — a glowing crystal block emitting blue-green light (`#88ccff`). Used as the foundation crystal for several other mods in this manual (the Enchanted Tree growth mechanic and the Crystal Anvil crafting bench).

### Features

- Custom block with glow light source
- Falls when unsupported (uses `Support.Down`)
- Multilingual: EN-US, ES, PT-BR

## Installation

1. Copy the mod folder to `%APPDATA%/Hytale/UserData/Mods/CreateACustomBlock/`
2. The folder name **must** be `CreateACustomBlock`
3. Start a Hytale server with the mod enabled

## Testing In-Game

```
/spawnitem Ore_Crystal_Glow
```
